# Septnn Doc

## 个人相关的文档，基于[DAUX](http://daux.io)

## 安装

1. **克隆**

```sh 
git clone https://github.com/sepntt/SeptnnDoc.git
```

2. **WEB服务**

```sh
cd septnndoc
php -S 127.0.0.1:81
```

3. **访问**
```sh
动态访问 ： `http://127.0.0.1:81`  
静态访问 ： 生成静态文件需要执行`./bin/daux generate`，然后访问`http://127.0.0.1:81/static`
```
