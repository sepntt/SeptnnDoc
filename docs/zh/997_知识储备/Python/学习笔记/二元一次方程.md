#问题：矩阵方程，求解二元一次方程

```
2x+y=1
3x+y=6
```

> 原理：矩阵乘法  
> A*X=b  
```
{2,1   
 3,1} 
* 
{x 
 y}
=
2*x+1*y
3*x+1*y
=
{1
 6}
```

> 设：矩阵方程  
```
A^(-1)*AX = A^(-1)*b
=
EX = A^(-1)b
=
X  = A^(-1)b
```

> 解1：矩阵的逆  
```
{2,1  
 3,1}
```

> 逆矩阵公式嵌套  
```
(2*1-1*3)/1
*
{1,-1 
 -3,2}
=
{-1,1
 3,-2}
```

> 验证解1：逆矩阵 单位矩阵  
```
{2,1
 3,1}
*
{-1,1
 3,-2}
=
2*-1+1*3=1
2*1+1*-2=0
3*-1*1*3=0
3*1+1*-2=1
=
{1,0
 0,1} # 单位矩阵
```

> 解2：  
```
X = 
{-1,1
 3,-2}
*
{1
  6}
=
-1*1+1*6=5
3*1+-2*-6=9
=
{5,-9}
```

> 答：  
```
2*5+-9=1
3*5+-9=6
```

> python代码

```python
import numpy as np

A = np.array([[2,1],[3,1]])
AI = np.linalg.inv(A)
b = np.array([[1],[6]])

X = np.dot(AI,b)

print(X)
```

> 延伸一个：  

> 问题：矩阵方程，求解三元一次方程  
```
2x+y+z=1
3x+y+z=6
x+y+4z=6
```
```python
import numpy as np
A = np.array([[2,1,1],[3,1,1],[1,1,4]])
AI = np.linalg.inv(A)
b = np.array([[1],[6],[6]])
X = np.dot(AI,b)
```
